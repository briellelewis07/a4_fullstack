What does "Full" in Full Stack mean?

	Full means that you develop both the front end, and the back end.
	The front end is what the user sees on what is called the client computer.
	The back end is what is on the server computer that delivers content to the front.

What does "Stack" in Full Stack mean?

	Stack means that several programs, a "stack" of programs is used.
	On the front end I use HTML5, CSS3, and JavaScript.
	On the back end I use PHP, and MySQL.
	And I also have a LAMP stack on my server computer.

What does LAMP mean?

	Lamp is a stack used for our web page server - remember, a server send web pages.
	LAMP stand for:
	L = Linux, which is the basis of our server operating system
	A = Apache, which is our web server
	M = MySQL, the database we use
	P = PHP, which is the programming language we use on the server side
  1. DataBase: Organized way to save data on server. LIFEBLOOD of all business, to track employees, customers, products, services, suppliers...
  2. MySQL: Number 2 among the most-used DB's in the entire world! Oracle is number 1 because it offers professional support, but it is a paid service. MySQL is FREE! The inventor of MySQL named this after his first daughter, her name is "My". This same inventor later created MariaDB, after his second daughter "Maria".
  3. SQL: Structured Query Language: MySQl is the name of the database, while SQL is the language you use inside of the database.
  4. PHP is a programming language that connects front end to back end. PHP reads HTML forms, then sends that form to the server.
